方法

方法名 参数 参数类型 说明
info ...args 不定参数 消息提示。第一个字符串参数作为消息文字text，第二个字符串参数作为消息标题title，第一个json对象参数作为options参数。
error ...args 不定参数 错误提示。第一个字符串参数作为消息文字text，第二个字符串参数作为消息标题title，第一个json对象参数作为options参数。
confirm ...args 不定参数 确认提示。第一个字符串参数作为消息文字text，第二个字符串参数作为消息标题title，第一个json对象参数作为options参数。
onConfirm fn（必须） Function 为弹出框添加“确认”回调，当传入的参数为非箭头函数时，该函数上下文（this）指向该PopService实例。
onCancel fn（必须） Function 为弹出框添加“取消”回调，当传入的参数为非箭头函数时，该函数上下文（this）指向该PopService实例。
onClose fn（必须） Function 为弹出框添加“关闭”(右上方关闭按钮点击后）回调，当传入的参数为非箭头函数时，该函数上下文（this）指向该PopService实例。
close 无 - 关闭提示框。

  • options说明

    属性名 类型 是否必须 说明
    title string 否 标题文字，默认值为“提示”
    text string 是 pop弹出框正文
    confirmButtonText string 否 “确定”按钮文字，默认值为“确定”
    cancelButtonText string 否 “取消”按钮文字，默认值为“取消”
    showConfirmButton boolean 否 是否显示“确定”按钮，默认为true
    showCancelButton boolean 否 是否显示“取消”按钮，默认为false
    confirmButtonType string 否 确定按钮类型，默认值为“info”，如值为“info”时，将会为“确定按钮”加上名为“btn-info”的class类，可选值包括“info”、“primary”、“success”、“warning”、“default”、“danger”、“dark”
    cancelButtonType string 否 取消按钮类型，默认值为“default”，如值为“info”时，将会为“取消按钮”加上名为“btn-info”的class类，可选值包括“info”、“primary”、“success”、“warning”、“default”、“danger”、“dark”
    closeOnConfirm boolean 否 是否点击“确定”按钮后关闭，默认值为true
    closeOnCancel boolean 否 是否点击“取消”按钮后关闭，默认值为true
    showLoaderOnConfirm boolean 否 是否在点击“确定”按钮后将按钮设为“loading”状态，默认值为false
    showLoaderOnCancel boolean 否 是否在点击“取消”按钮后将按钮设为“loading”状态，默认值为false
    confirmLoaderContent string 否 “确定”按钮loading时按钮内容，默认使用按钮文字
    cancelLoaderContent string 否 “取消”按钮loading时按钮内容，默认使用按钮文字
    textAlign string 否 pop正文的对齐方式，可选值“left”、“center”,默认为“left”
    iconClass string 否 pop正文开始位置图标的class类，可通过此参数为弹出框添加字体图标
