Layout

示例

如本站整体布局所示

类名 模块 selector 属性 说明
RootContainerComponent LayoutModule root-container

@Input() headerFixed:boolean;//是否顶部固定

@Input() asideFixed:boolean;//是否左侧固定

@Input() asideFolded:boolean;//是否收起左侧菜单

@Input() offScreen:boolean;//是否处于小屏幕时展开菜单（此时遮住屏幕）

 是整体布局的根组件，控制页面整体布局
HeaderComponent LayoutModule header @Input() theme:string='';//主题 当前可选：black、cerulean，默认black 顶部布局组件
HeaderLeftComponent LayoutModule header-left @Input() theme:string='';//主题 当前可选：black、cerulean，默认black 顶部左边，header的子组件。
HeaderRightComponent LayoutModule header-right @Input() theme:string='';//主题 当前可选：black、cerulean、white，默认black 顶部右边，header的子组件。
AsideLeftComponent LayoutModule aside-left @Input() theme:string='';//主题 当前可选：black、cerulean-outline、white，默认black 左侧边栏布局组件。
