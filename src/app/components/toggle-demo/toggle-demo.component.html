属性

属性名 类型 输入/输出类型 说明
value Input boolean 输入值。默认false。
valueChange Output EventEmitter< any > 输入值改变事件，触发时传入参数为value的值。
size Input string 组件尺寸。默认md,可选：lg、md、sm。
disabled Input any 绑定在组件内input元素上disabled属性的值。
type Input string （背景）类型，默认为info ，可选 primary、success、danger、warning、info。
styleClass Input string 绑定在组件内最外层元素上的css类。
auto Input boolean 是否点击后切换，默认为true。
action Output EventEmitter< any > auto不为真时点击触发的动作，传入参数为value。
name Input string 绑定在组件内input元素上name属性的值。
