switchcomponent

示例

@Input

属性名 类型 说明
value boolean 输入值。默认false。
size string 组件尺寸。默认md,可选：lg、md、sm。
disabled any 绑定在组件内input元素上disabled属性的值。
type string （背景）类型，默认为info ，可选 primary、success、danger、warning、info。
styleClass string 绑定在组件内最外层元素上的css类。
auto boolean 是否点击后切换，默认为true。
name string 绑定在组件内input元素上name属性的值。

@Output

属性名 类型 说明
valueChange EventEmitter < any> 输入值改变事件，触发时传入参数为value的值。
action EventEmitter < any> auto不为真时点击触发的动作，传入参数为value。
