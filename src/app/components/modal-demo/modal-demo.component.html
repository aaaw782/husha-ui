@Input (modal-header,modal-body,modal-footer仅支持stylClass属性)

属性名 类型 说明
visible boolean 控制显示和隐藏
overflow boolean 是否溢出屏幕，默认为false。
animated boolean 是否有过渡动画，默认为true。
size string 模态框尺寸，默认为md，可选lg、md、sm。
fullHeight boolean 是否默认撑满屏幕高度；默认为false。
disabled boolean 是否禁用（有遮罩）；默认为false。
outClickClose boolean 是否点击空白处关闭；默认为false。
showCloseButton boolean 是否显示关闭按钮；默认为true。
width number 模态框宽度。
styleClass string 添加到组件内最外层元素上的css类。

@Output

属性名 类型 说明
visibleChange EventEmitter < any> visible改变事件；触发时传入参数为visible。
onOpen EventEmitter < any> 模态框从隐藏到显示完成时的事件；触发时传入参数为visible。
onClose EventEmitter < any> 模态框关闭事件；触发时传入参数为visible。
ModalModule

关系：

1.modal是父组件，modal-header、modal-body、modal-footer是子组件；

2.子组件应被包含在父组件中。

两种使用方式：

1.通过属性visible双向绑定控制显示/隐藏；

2.通过声明模版变量（指向组件实例），调用open/close方法控制显示/隐藏。
