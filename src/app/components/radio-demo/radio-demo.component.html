示例

radio={{radio}} radio={{radio}} radio={{radio}} radio={{radio}} radio={{radio}} radio={{radio}}

@Input

属性名 类型 说明
key any 绑定的key值。
value any key对应的值。
size string 组件尺寸。默认md,可选：lg、md、sm。
display string 显示方式。默认为行内，可选block（块级）
disabled any 绑定在组件内input元素上disabled属性的值，当该值为真时组件将被禁用。
customBackground string 勾选时的背景，值为颜色值，如rgba(0,0,0,1)、#000000。
styleClass string 绑定在组件内最外层元素上的css类。
name string 绑定在组件内input元素上name属性的值。

@Output

属性名 类型 说明
keyChange EventEmitter < any> key值改变事件，触发时传入参数为key的值。
